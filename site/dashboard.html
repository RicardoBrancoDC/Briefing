Este painel usa exclusivamente os dados que já são processados pelo workflow atual. O script principal não foi alterado. A distinção de alertas DCA não aparece aqui porque essa informação não está disponível diretamente no CAP publicado. Por isso, o quarto card foi ajustado para mostrar a quantidade de alertas classificados como Extremo.

Se depois você quiser, a gente pode transformar este dashboard na página principal do site, sem perder a página antiga de mapas e imagens.
